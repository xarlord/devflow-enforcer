# UI/UX Agents

This directory contains agents specializing in user interface and experience design.

## Agents

| Agent | Description | Phase |
|-------|-------------|-------|
| `ui-ux-designer-agent.md` | UI/UX design specialist | Design phases |
| `accessibility-expert-agent.md` | WCAG compliance and accessibility | Testing phases |
| `frontend-developer-agent.md` | Frontend implementation | Development |

## Responsibilities

- UI/UX design review and feedback
- Accessibility auditing (WCAG 2.1 AA)
- Design system creation and maintenance
- Frontend component development
- User experience optimization

## Quality Metrics

| Metric | Target |
|--------|--------|
| Accessibility Score | 100% WCAG 2.1 AA |
| Design System Coverage | 100% components |
| UI Review Pass Rate | 100% |
